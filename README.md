# python-for-students
Програмне забезпечення для лабораторних робіт з дисциплін Електричної інженерії

Цей репозиторій містить розроблені викладачем Циганчук Василь Васильович програмні комплекси для використання у навчальному процесі.

## Перелік програм:
* `Професійний калькулятор вітки змінного струму.py` — Розрахунок і побудова векторних діанрам напруг і струмів в колі змінного струму з послідовним включенням RLC-елементів в різній комбінації.


## Інструкція для студентів:
1. Завантажте файл на комп'ютер.
2. Запустіть у середовищі IDLE, PyCharm або VS Code.
3. Введіть індивідуальні вхідні дані згідно з методичкою.
